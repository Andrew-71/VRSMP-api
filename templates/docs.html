The VRSMP API docs

Hello, and welcome to VillagerRights SMP's official api docs.

This system allows you to get info about server status, statistics of players and more!

How to call the API in Python

Some of you may be wondering, "How do I call this API?". Good question!

You do it the same way you call other APIs, here is an example in Python:


        from requests import request  # First, import the module used to access web urls

        request_response = request("GET", "api.villagerrights.xyz/status/current_players")  # Then make the API call you need

        data = request_response.json()  # Convert the response to JSON format. JSON is identical to Python dictionary.
